Introduction

JavaScript is a lightweight, interpreted programming language.

It is designed for creating network-centric applications.

It is complementary to and integrated with Java.

JavaScript is very easy to implement because it is integrated with HTML.

console.log("Hello, JavaScript!");
  • Client-side scripting
  • Object-oriented features
  • Event-driven
  • Lightweight
  • Embedded in HTML
What you should already know

You should understand basic programming concepts.

Familiarity with HTML and CSS is helpful.

JavaScript and Java

JavaScript and Java are different languages.

Java is a programming language, JavaScript is a scripting language.

Hello world

To get started with JavaScript, write your first program:

Variables

You use variables to store data values.

Use `let`, `const`, or `var` to declare variables.

let x = 10; const y = 5; var z = 15;
